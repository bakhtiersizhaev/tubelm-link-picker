English: install for beginners

Install TubeLM as an unpacked Chrome extension: download the ZIP from GitHub, unzip it, load the extracted folder in Chrome, then use it on YouTube.

1. Download and unzip

  1. Open the TubeLM GitHub page.
  2. Click Code, then Download ZIP, or use the direct ZIP link.
  3. Find tubelm-link-picker-main.zip in Downloads.
  4. Extract the ZIP file.
  5. Open the extracted folder and make sure it contains manifest.json.

2. Load in Chrome

  1. Open Chrome and go to chrome://extensions.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select the extracted TubeLM folder, not the ZIP file.
  5. Pin TubeLM from the puzzle-piece extensions menu.

3. Use on YouTube

  1. Open YouTube.
  2. Go to a channel, playlist, search results page, feed, or Shorts page.
  3. Select videos with the TubeLM checkboxes.
  4. Open TubeLM and click Copy URLs.
  5. Paste the links into NotebookLM as sources.

Troubleshooting

  • If Chrome cannot load it, select the folder that contains manifest.json.
  • If checkboxes are missing, reload the YouTube tab.
  • If Chrome shows a developer-mode message, that is normal for unpacked extensions.

Русский: установка для новичков

TubeLM устанавливается как распакованное расширение Chrome: скачайте ZIP с GitHub, распакуйте архив, загрузите папку в Chrome и используйте расширение на YouTube.

1. Скачайте и распакуйте

  1. Откройте страницу TubeLM на GitHub.
  2. Нажмите Code, затем Download ZIP, или используйте прямую ссылку на ZIP.
  3. Найдите файл tubelm-link-picker-main.zip в папке загрузок.
  4. Распакуйте ZIP-архив.
  5. Откройте распакованную папку и проверьте, что внутри есть manifest.json.

2. Загрузите в Chrome

  1. Откройте Chrome и перейдите на chrome://extensions.
  2. Включите Режим разработчика.
  3. Нажмите Загрузить распакованное.
  4. Выберите распакованную папку TubeLM, не ZIP-файл.
  5. Закрепите TubeLM через иконку пазла в панели расширений.

3. Используйте на YouTube

  1. Откройте YouTube.
  2. Перейдите на канал, плейлист, страницу поиска, ленту или Shorts.
  3. Отметьте нужные видео чекбоксами TubeLM.
  4. Откройте TubeLM и нажмите Copy URLs.
  5. Вставьте ссылки в NotebookLM как источники.

Если не работает

  • Если Chrome не загружает расширение, выберите папку с manifest.json.
  • Если чекбоксы не появились, перезагрузите вкладку YouTube.
  • Если Chrome показывает сообщение про режим разработчика, это нормально для распакованных расширений.

中文：新手安装教程

TubeLM 作为已解压的 Chrome 扩展安装：从 GitHub 下载 ZIP，解压文件夹，在 Chrome 中加载文件夹，然后在 YouTube 上选择视频链接。

1. 下载并解压

  1. 打开 TubeLM GitHub 页面。
  2. 点击 Code，再点击 Download ZIP，也可以使用 ZIP 直链。
  3. 在下载文件夹中找到 tubelm-link-picker-main.zip。
  4. 解压 ZIP 文件。
  5. 打开解压后的文件夹，确认里面有 manifest.json。

2. 在 Chrome 中加载

  1. 打开 Chrome，进入 chrome://extensions。
  2. 开启 Developer mode（开发者模式）。
  3. 点击 Load unpacked。
  4. 选择解压后的 TubeLM 文件夹，不要选择 ZIP 文件。
  5. 通过拼图图标菜单把 TubeLM 固定到工具栏。

3. 在 YouTube 上使用

  1. 打开 YouTube。
  2. 进入频道、播放列表、搜索结果页、首页推荐流或 Shorts 页面。
  3. 用 TubeLM 的复选框选择需要的视频。
  4. 打开 TubeLM，点击 Copy URLs。
  5. 把链接粘贴到 NotebookLM 作为 sources。

无法使用时

  • 如果 Chrome 无法加载扩展，请选择包含 manifest.json 的文件夹。
  • 如果 YouTube 没有显示复选框，请刷新 YouTube 标签页。
  • 如果 Chrome 显示开发者模式提示，这是已解压扩展的正常现象。

Español: instalación para principiantes

TubeLM se instala como una extensión descomprimida de Chrome: descarga el ZIP desde GitHub, descomprímelo, carga la carpeta en Chrome y úsalo en YouTube.

1. Descarga y descomprime

  1. Abre la página de TubeLM en GitHub.
  2. Haz clic en Code, luego en Download ZIP, o usa el enlace directo al ZIP.
  3. Busca tubelm-link-picker-main.zip en Descargas.
  4. Descomprime el archivo ZIP.
  5. Abre la carpeta extraída y comprueba que contiene manifest.json.

2. Carga en Chrome

  1. Abre Chrome y entra en chrome://extensions.
  2. Activa Developer mode o Modo de desarrollador.
  3. Haz clic en Load unpacked o Cargar descomprimida.
  4. Selecciona la carpeta TubeLM descomprimida, no el ZIP.
  5. Fija TubeLM desde el menú de extensiones con icono de rompecabezas.

3. Usa en YouTube

  1. Abre YouTube.
  2. Entra en un canal, playlist, resultados de búsqueda, inicio o Shorts.
  3. Selecciona videos con los checkboxes de TubeLM.
  4. Abre TubeLM y haz clic en Copy URLs.
  5. Pega los enlaces en NotebookLM como fuentes.

Si algo no funciona

  • Si Chrome no puede cargar la extensión, selecciona la carpeta que contiene manifest.json.
  • Si no aparecen checkboxes, recarga la pestaña de YouTube.
  • Si Chrome muestra un aviso de modo desarrollador, es normal para extensiones descomprimidas.
